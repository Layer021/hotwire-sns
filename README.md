# Hotwire SNS Sample
Hotwireを使ったSNSのサンプルアプリケーションです。このアプリケーションには以下の機能があります。
- ログイン機能
- 投稿機能
- いいね機能
- フォロー機能
- ユーザー検索機能

## 使用したライブラリなど
- Docker
- Ruby on Rails
- Hotwire
- Devise
- Tailwind CSS + daisyUI
